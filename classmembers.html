クラスメンバ
製品の概要 > 更新履歴 > 2.0J > クラスメンバ

PDF for .NET 2.0J では、以下のクラスメンバが追加されました。

PDF フォーム

メンバ 説明
C1PdfDocument.AddField メソッド フォームフィールドを現在のページに追加します。
PdfCheckBox クラス チェックボックスを表します。
PdfComboBox クラス コンボボックスを表します。
PdfListBox クラス リストボックスを表します。
PdfPushButton クラス ボタンを表します。
PdfRadioButton クラス ラジオボタンを表します。
PdfTextBox クラス テキストボックスを表します。
PdfFieldBase クラス 他のすべての PDF フォーム フィールドクラス（フォームフィールド）の派生元となる抽象基本クラス。

署名

メンバ 説明
PdfSignature クラス PDF フォームの電子署名フィールドを表します。

HTML 描画

メンバ 説明
C1PdfDocument.DrawStringHtml メソッド HTML 文字列を表示します。
C1PdfDocument.MeasureStringHtml メソッド 指定された条件で表示されたときの HTML 文字列を計測します。

注釈

メンバ 説明
C1PdfDocument.AddNote メソッド 注釈項目を現在のページに追加します。
