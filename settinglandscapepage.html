用紙の方向を設定するには、Landscape プロパティを使用します。このプロパティは、デザイナまたはコードのいずれかで設定できます。

デザイナの場合

プロパティウィンドウで Landscape プロパティを探し、True に設定します .

コードの場合

次のコードを Form_Load イベントに追加し、Landscape プロパティを設定します。

Visual Basic コードの書き方

Visual Basic
コードのコピー 
' 横の用紙方向を使用します。
C1PdfDocument1.Landscape = True
 ' ドキュメントをファイルに保存します。
C1PdfDocument1.Save("c:\mydoc.pdf")

C# コードの書き方

C#
コードのコピー 
// 横の用紙方向を使用します。
c1PdfDocument1.Landscape = true;
// ドキュメントをファイルに保存します。
c1PdfDocument1.Save(@"c:\mydoc.pdf");
