ページサイズを設定するには、PaperKind プロパティを使用します。このプロパティは、デザイナまたはコードのいずれかで設定できます。

デザイナの場合

プロパティウィンドウで PaperKind プロパティを探し、新しい用紙サイズを選択します。

コードの場合

次のコードを Form_Load イベントに追加し、PaperKind プロパティをリーガルサイズ用紙に設定します。

Visual Basic コードの書き方

Visual Basic
コードのコピー 
' Legal サイズ用紙を使用します。
C1PdfDocument1.PaperKind = Printing.PaperKind.Legal
 ' ドキュメントをファイルに保存します。
C1PdfDocument1.Save("c:\mydoc.pdf")
 ' Save the document to a file.
pdf.Save("c:\mydoc.pdf")

C# コードの書き方

C#
コードのコピー 
// Legal サイズ用紙を使用します。
c1PdfDocument1.PaperKind = Printing.PaperKind.Legal;
 // ドキュメントをファイルに保存します。
c1PdfDocument1.Save(@"c:\mydoc.pdf");
 // Save the document to a file.
pdf.Save(@"c:\mydoc.pdf");
